Review App — Visual Quick Reference

One page. The core loop for correcting a trip's text & audio, then submitting.

The loop, step by step

  1. Log in with your username & password (link from David).
  2. Open a trip from your list. Pinned trips at the top are urgent.
  3. Mandarin only: listen to V2 and V3 and pick the better voice (once per trip).
  4. Fix the text of each part — trip title, description, and each scene's title / narration / question / options.
  5. Regenerate the audio for what you changed → listen → Combine.
  6. Play the whole clip (the bar must fill) → the Done button unlocks → click it.
  7. When every part is Done → Submit. David approves.
  8. Something wrong you can't fix? Report a problem (write in your language) or comment + flag "Edit required".
  9. Need help? The ? button in the top bar opens this page and your full guide.

The buttons under each audio part

↶ Undo ↷ Redo Regenerate All Generate from edit ① Regenerate highlighted ⑤ Fix pronunciation… ⑤ Trim highlighted noise ⑤ Insert 1s ⑤ Insert 0.5s ⑤ Remove 1s ⑤ Remove 0.5s ⑤ Trim end silence Combine ② Create new ③ Report a problem ④
Where to highlight for the ⑤ tools — English: the narration text · Japanese: the kana (last) line · Mandarin: the 简体 Simplified box.
1
Generate from edit — re-records only the words you changed; the rest of the original recording is kept. Enabled once you've edited the spoken text (Japanese: the kana line; Mandarin: the Simplified line — the app reminds you if you edited another line).
2
Combine — keeps the new take you just auditioned. Do this after you're happy with the sound.
3
Create new — attach a fresh take with a note for David when the auto tools can't get it right.
4
Report a problem — describe the issue in your language; the current text + audio are saved automatically. Track replies under Bug reports in the top bar.
5
Highlight tools — fix a spot without changing the text: highlight it, then re-record just that phrase (Regenerate highlighted), speak different text there (Fix pronunciation… — e.g. a phonetic spelling; the on-screen text is unchanged), clean a noise (Trim highlighted noise — also available on question/answer audio, highlighting in that field's own box), or click at a pause to lengthen / shorten it by 1s or 0.5s (Insert / Remove 1s / 0.5s — a natural pause is always kept). If a spot can't be done safely the app says so; sometimes it re-records the whole line instead and tells you — re-listen fully.

Two things that trip people up

Mandarin

You edit four scripts — 繁體 / 简体 / 注音 / English. The 简体 (Simplified) line is what the voice speaks. You never type pinyin — it's generated from Zhuyin.

Japanese

Each narration has a Kanji line and a Kana line. The voice speaks the Kana line. To change the audio you must edit the Kana — editing only the Kanji won't change the sound.

Marking "Done" needs a full listen. The Done button stays locked until you've played the whole clip (seeking to the end doesn't count) — this is deliberate, so nothing ships un-checked.
